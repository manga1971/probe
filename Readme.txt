# 🎙️ probem – Taskuri transformate în acțiuni clare

Probe este o aplicație  care transformă ideile și instrucțiunile tale, rapoarte sau fișe structurate – totul local, fără costuri mari de procesare în cloud.

## 🔥 Beneficii principale

- 🎤 direct din aplicație
- 📝 local 
- 📊 Analiză + organizare în format tabelar sau PDF
- 📨 Export ușor în email/PDF pentru delegare către echipă
- 💡 Ideal pentru freelanceri activi

## 🛠️ Tehnologii folosite

- `JavaScript`, `HTML`, `Tailwind` – interfață web
- `Web Audio API` – pentru captarea audio
- `pdf-lib`, `html2pdf.js` – pentru export
- `localStorage` / `IndexDB` – salvare locală
